About This Application

Stratux FIS-B Weather Monitor — a single-page web application for monitoring FIS-B weather products, graphical radar/icing/turbulence data, tower coverage, ownship tracking, and winds aloft with animation.

Built with reference to:

  • cyoung/stratux — Stratux ADS-B receiver: WebSocket API, weather/GDL90/tower endpoints, GDL90 protocol
  • rand-projects/fisb-decode — FIS-B product decoding reference: DLAC text, TWGO, graphical products, SUA, NOTAM, CRL, block numbering, RLE schemes

Additional sources:

  • FAA Order 7900.5 / FMH-1 — METAR/TAF weather phenomena codes, sky condition codes, remark groups (SLP, T-group, precip, pressure tendency, cloud types, lightning), wind barb symbology
  • FAA AC 00-45 (Aviation Weather Services) — PIREP field codes (/OV /TM /FL /TP /SK /WX /TA /WV /TB /IC /RM), turbulence and icing intensity codes
  • FAA JO 7340.2 (Contractions) — NOTAM abbreviation expansion (TWR, CLSD, RWY, OBST, MALSR, etc.)
  • RTCA DO-267A / DO-358B — GDL90 ICD, DLAC character encoding, FIS-B product registry, graphical overlay specifications, SUA type/status codes
  • Natural Earth 10m Admin 1 (States/Provinces) — CONUS state boundary polygons (naturalearthdata.com, public domain)
